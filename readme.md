#Simple Android Oscilloscope

Yet another android oscilloscope.

Sensors:
- Audio Sensor (dB)
- Light Sensor (lx)
- Magnetic Sensor (uT)
- Temperature Sensor (°C)
- Accelerometer Sensor (m·s^-2)
